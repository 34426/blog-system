### 一、项目介绍

语言：java

运行工具：idea或eclipse vscode 数据库：mysql

基于SpringBoot MyBatis JQuery html实现的博客网站

博客网站 

三个角色：游客 用户 管理员

游客可以浏览文章， 游客可以登录注册成用户，发布文章 管理自己的文章，评论和回复，点赞评论回复文章等 

管理员可以对整个系统用户管理，文章管理，分类管理，角色权限管理，评论管理等等
